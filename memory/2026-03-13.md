## ProspectBot: Calendar Events System Fixed & Enhanced

**Problem:** Tyler couldn't see any calendar events for customers — "No IndoorMedia Events Found"

**Root causes:**
1. Bot was calling `gog calendar list` without date range → only showed next few days
2. PDF parser couldn't handle multi-word store names ("Fred Meyer-0208", "Quality Food Center-0206") due to regex `[\w]+-\d+`
3. 9 of 18 contracts never had calendar events created

**Fixes applied:**
- Updated bot calendar search: `gog calendar list --from TODAY --to +400days` (line ~6145)
- Fixed PDF parser regex: `[\w][\w\s]*?-\d+` to allow spaces in store names

- Created 63 new calendar events for 9 missing contracts
