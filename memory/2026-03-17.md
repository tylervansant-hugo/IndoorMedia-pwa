# Session Notes - March 17, 2026

## Bugs Fixed

### Calendar Appointments (Commits: 1a03be3)
**Issues:**
- Rep appointments were booking on Tyler's calendar (primary) instead of rep's own calendar
- Timezone was wrong (3 PM became 4 PM due to missing timezone spec)

**Solution:**
- Changed `gog calendar create primary` → `gog calendar create {rep_email}` to book on rep's calendar
- Added timezone: `America/Los_Angeles` to time objects before sending to gog
- Tyler is auto-invited as attendee to all appointments

**Result:** When Adan books 3 PM meeting, it appears on Adan's calendar at 3 PM with Tyler invited.

### Testimonials Hanging / Not Loading (Commits: 9986391, c753977)
**Initial Problem:** User clicked "Testimonials" button and it hung forever on "loading"

**Root Cause:** Old complex category + geography matching was broken; TESTIMONIALS cache empty at startup

**Complete Rewrite:**
- Changed from complex category/geo logic to **simple keyword search with fallback chain**
- Now extracts primary keyword from business name (e.g., "Kenji's Ramen" → "Ramen")
- Searches primary keyword first
- If <3 results, tries expanded keywords (Ramen → [Sushi, Teriyaki, Japanese, Asian, Noodles])
- Returns ~3 unique results (no duplicates)

**Result:** Instant, reliable testimonials. No more hanging.
