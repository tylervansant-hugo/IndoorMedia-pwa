# March 23, 2026 — imPro PWA Build Session

## Session Summary
Built comprehensive quote builder tool for imPro Sales Portal PWA. Transformed cart from generic shopping cart to a full **Build Quote** workflow with product selection, store selection, payment plans, and cycle/shipment data.

## Major Features Completed

### 1. Build Quote Tool (Cart Tab Redesign)
- **Multi-step workflow:**
  - Select product type (Register Tape, Cartvertising, Digital products)
  - Choose store (with search, address, cycle display)
  - Select payment plan (for tape only) or pin count (for DigitalBoost)
  - Review and export quote as CSV

- **Products supported:**
  - Register Tape (Co-Op, Exclusive, Contractors) with 4 payment plans each
  - Cartvertising (8 packages: 20%, 40%, 60%, 80%, 100%, 200%, Header 50%, Header 100%)
  - DigitalBoost (1-5 pins, $3,600/pin + $395 production)
  - FindLocal ($695/location)
  - ReviewBoost ($695)
  - LoyaltyBoost ($3,600/year)

### 2. Store Selection & Cycle Data
- All products now require store selection (except digital can use map area)
- Store cards show:
  - Full store name, city, state
  - **Complete address**
  - **Cycle (A/B/C)** with red badge
  - Store number
  - Single/Double ad pricing
  - **Next cycle launch date** (green text)

- DigitalBoost and FindLocal have "🗺️ Choose Area on Map Instead" blue button

### 3. Cycle Launch Dates (Zone 07)
- **A cycles:** Jan 7, Apr 7, Jul 7, Oct 7
- **B cycles:** Feb 7, May 7, Aug 7, Nov 7
- **C cycles:** Mar 7, Jun 7, Sep 7, Dec 7
- Logic auto-calculates next launch for any cycle letter
- Shows on store selector and quote items

### 4. Zone Shipment Data (28 zones)
- Created `/pwa/public/data/zones.json` with:
  - All 28 national zones + states + chains
  - "In Stores" day of month for each zone (from May 2022 RTUI chart)
  - Zone 07Z marked as Tyler's territory with cycle schedule detail
  - Audit schedule guidance (45-day window from shipment)

- Created `/pwa/src/lib/auditHelper.js` for:
  - Extract zone from store number (e.g., `FME07Z-0236` → `07Z`)
  - Calculate audit due date (delivery + 45 days)
  - Get shipment window per zone

### 5. Cart Icon Header
- White semi-transparent background (matches day/night theme like logout)
- Red badge shows item count
- Tap to open Build Quote tab
- Removed Cart from bottom nav

### 6. Quote Export
- CSV export with columns: Product, Store, Store #, Address, Cycle, Plan, Price, Date
- Downloaded as `IndoorMedia_Quote_YYYY-MM-DD.csv`
