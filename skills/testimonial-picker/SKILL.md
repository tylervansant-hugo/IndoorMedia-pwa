# testimonial-picker

**Curated testimonial packages for sales appointments.**

Given a business type and city, pulls a matching video testimonial, 3-4 strong written testimonials in the same category, and a nearby written testimonial.

## What It Does

When Tyler has an appointment with a business (e.g., "pizza place in Longview" or "nail salon in Bend"), this skill returns:

**A) 1 video testimonial** - A video from Google Drive in the same or similar category
- Prioritizes videos from the "flat" folder (more recent, OR/WA region)
- Falls back to categorized folder videos

**B) 3-4 written testimonials** - Top performers in the same category
- High redemption numbers (perMonth, perWeek)
- Same or closely related category
- Bonus points for ROI mentions, specific revenue numbers
- Links to PDF testimonials

**C) 1 nearby testimonial** - Geographically close to the appointment city
- Same state preferred (OR/WA priority)
- Different from the main matches
- Useful for showing local success

## Usage

### Command line:

```bash
# Simple usage
python3 skills/testimonial-picker/scripts/testimonial_picker.py "pizza place" "Longview"

# With state
python3 skills/testimonial-picker/scripts/testimonial_picker.py "Thai restaurant" "Portland" --state "OR"

# Explicit category
python3 skills/testimonial-picker/scripts/testimonial_picker.py --category "Pizza" --city "Longview" --state "WA"
```

### In conversation:

**Tyler:** "I have an appointment with a pizza place in Longview"

**Agent:** _Runs testimonial picker and returns formatted results ready to paste in Telegram_
