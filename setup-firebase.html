🔥 Firebase Setup for imPro

This connects the app to Firebase so you can see all reps' activity across devices.

Step 1: Create Firebase Project

  1. Go to Firebase Console
  2. Click "Create a project" → Name it "impro-sales"
  3. Disable Google Analytics (not needed) → Create
  4. Click "Web" icon (</>) to add a web app
  5. Name it "imPro PWA" → Register
  6. Copy the config values below 👇

Step 2: Enable Firestore

  1. In Firebase Console → Build → Firestore Database
  2. Click "Create database"
  3. Choose "Start in test mode" (we'll secure later)
  4. Select region: us-west1 → Create
