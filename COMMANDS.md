# Commands & Quick Reference

## @IndoorMediaProspectBot 🎯
Search for prospects near stores by category

- **/start** - Start prospect search
- **/help** - How to use the bot
- **/examples** - Example cities & stores
- **/city** - Search by city name
- **/store** - Search by store number
- **/reset** - Reset current search

**Usage:**
- Send city name: `Honolulu`
- Send store number: `FME07Z-0236`
- Follow prompts for category → subcategory → results

## @IndoorMediaRatesBot 💰
Look up store pricing & payment plans

- **/start** - Start pricing lookup
- **/help** - How to use the bot
- **/cities** - List all available cities
- **/chains** - List all available chains
- **/reset** - Reset search

**Usage:**
- Send store number: `FME07Z-0236`
- Send city + chain: `Denver Kroger`
- Tap buttons to toggle Single/Double ad pricing

## Quick Tips
- City search: Supports partial matches (e.g., `Denver`, `Portland`, `Henderson`)
- Store format: `CHAIN##Z-####` (e.g., `SAF07Y-0415`, `FME07Z-0236`)
- Cycles: A, B, or C (from store data, not store number)
- Addresses: Clickable in prospect results (tap to copy)
- Navigation: Back buttons to return to categories/cities without re-searching
